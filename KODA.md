## Обзор проекта

Данный проект представляет собой внешнюю обработку для платформы 1С:Предприятие, предназначенную для анализа прав доступа ролей и профилей к объектам метаданных конфигурации. Результаты анализа могут быть выгружены в формате Markdown. Обработка позволяет определить, какие роли/профили имеют доступ к указанному объекту, или какие объекты доступны указанной роли/профилю, а также информирует о наличии ограничений RLS. **Обработка также может работать с подключенным расширением ИИкона https://github.com/andromanpro/1c-ai-connector

## Технологии

*   **Язык программирования:** 1С (BSL - 1C:Enterprise Script Language).
*   **Платформа:** 1С:Предприятие.

## Структура проекта

Проект организован в соответствии со стандартной структурой внешних обработок 1С при выгрузке в файлы. Основные компоненты и их предназначение:

*   `ПродвинутыйАнализПрав/Ext/ObjectModule.bsl`: Содержит модуль объекта обработки. Это основной бэкенд-код обработки, где реализована логика анализа прав доступа.
*   `ПродвинутыйАнализПрав/Forms/Форма/Ext/Form/Module.bsl`: Содержит модуль формы обработки. Здесь находится код, отвечающий за интерактивность пользовательского интерфейса, обработчики событий формы и команд, а также логика отображения результатов и взаимодействия с пользователем.
*   `ПродвинутыйАнализПрав/Forms/Форма.xml`: XML-описание, описывающее свойства и метаданные формы обработки "Форма".
*   `ОписаниеФормы.txt`: Подробная инструкция в текстовом формате по созданию данной обработки в конфигураторе 1С. Описывает реквизиты, команды и структуру элементов формы. Этот файл полезен для ручного воссоздания или понимания структуры обработки.
*   `ПродвинутыйАнализПрав.epf`: Скомпилированный бинарный файл внешней обработки для 1С:Предприятия. Это готовый к использованию файл, который можно открыть в режиме "Предприятие" 1С.
*   `ПродвинутыйАнализПрав.xml`: Основной XML-файл, описывающий метаданные всей обработки. Этот файл, в совокупности с поддиректорией `ПродвинутыйАнализПрав/` и ее содержимым, позволяет восстановить обработку в конфигураторе 1С с помощью функции "Загрузить из файлов".

## Использование и восстановление

Для использования обработки достаточно открыть файл `ПродвинутыйАнализПрав.epf` в режиме "Предприятие" 1С через меню "Файл" -> "Открыть...".

Если необходимо восстановить или произвести доработку обработки в конфигураторе, это можно сделать следующим образом:

1.  Откройте конфигуратор 1С.
2.  В меню "Файл" выберите "Загрузить из файлов...".
3.  Укажите файл `ПродвинутыйАнализПрав.xml`. Конфигуратор автоматически подтянет все связанные файлы из поддиректории `ПродвинутыйАнализПрав/` и восстановит обработку.

## Правила разработки

*   **Стиль кодирования:** Используется русскоязычное именование переменных, процедур и функций. Комментарии также на русском языке. Присутствуют области кода (`#Область ... #КонецОбласти`) для структурирования.
*   **Тестирование:** В предоставленном коде явных юнит-тестов не обнаружено. Ожидается, что тестирование осуществляется через функциональную проверку в среде 1С:Предприятия.
